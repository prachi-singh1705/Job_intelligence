# Job Intelligence System

A Python-based job intelligence platform that scrapes job data from
LinkedIn and Internshala, stores it in a database, and displays insights
via a Streamlit dashboard.

## Features
- Job scraping
- Data storage using SQLite
- Streamlit dashboard
- Multi-threaded execution

## Tech Stack
- Python
- Streamlit
- SQLite
- Web Scraping

## How to Run
```bash
pip install -r requirements.txt
python run_all.py
